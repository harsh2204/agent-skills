# agent-skills

Agent Skills for [harsh2204/agent-skills](https://github.com/harsh2204/agent-skills).

## Install

Add a skill using the direct path to its folder in this repo:

```bash
npx skills add https://github.com/harsh2204/agent-skills/tree/main/picasso
```
